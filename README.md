##  Credit Card Spending Data Analysis

This project was developed to analyze credit card users' spending data and obtain meaningful insights into user behavior.

## Phase 1: Data visualization with Python

### Main Objectives

-To systematically analyze credit card spending data.
-To obtain data-driven insights for customer segmentation.
-To identify geographical, temporal, and demographic spending patterns.
-To provide action plans for business development and marketing strategies.

### Technologies Used

-Python 3.x - Main programming language
-Pandas - Data manipulation and analysis
-Matplotlib - Basic visualizations
-Seaborn - Statistical data visualization
-NumPy - Numerical calculations
-Google Colab - Development environment

### Data Sets

users.csv - User demographic information

User ID, gender, annual income information
Basic data for customer profile analysis

transactions.csv - Credit card transaction details

Spending amounts, date/time, city information
Transaction data for behavioral analysis
